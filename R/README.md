# R Analysis
## Cyclistic Bike-Share Case Study

This section demonstrates statistical analysis and visualization using R with the tidyverse ecosystem.

**Status:** In development

### Planned Deliverables
- Data profiling report using `skimr` and `DataExplorer`
- Core analysis replicating Power BI findings using `dplyr`
- Visualizations using `ggplot2`
- Peak concurrent rides / fleet utilization analysis using interval overlap algorithm

### Why R
R with the tidyverse provides a code-first approach to the same analytical questions answered visually in Power BI. This section demonstrates the ability to produce reproducible, documented analysis in a scripting environment — complementing the dashboard approach of the Power BI section.
