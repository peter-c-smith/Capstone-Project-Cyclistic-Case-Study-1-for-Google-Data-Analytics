# DuckDB Analysis
## Cyclistic Bike-Share Case Study

This section demonstrates analytical SQL using DuckDB — a modern, lightweight in-process analytical database that queries CSV files directly without requiring a server or data import step.

**Status:** Planned

### Planned Deliverables
- Core analytical queries replicating the Cyclistic analysis in SQL
- Comparison notes contrasting DuckDB with the SQL Server approach
- Runnable `.sql` files — DuckDB can be installed locally and queries run directly against the source CSV files

### Why DuckDB
DuckDB is gaining significant traction in the data analytics community as a fast, portable alternative to traditional RDBMS for analytical workloads. Its ability to query CSV and Parquet files directly makes it ideal for exploratory analysis without infrastructure overhead. The SQL syntax is nearly identical to the SQL Server queries in this project, making direct comparison straightforward.
